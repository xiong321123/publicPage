从高三90多到高考147，她背单词的秘密

从高三90多到高考147，让我们来揭开背单词的骗局

1.引言​

膨胀螺栓去年7月份加入我们，当时英语90多分，今年高考英语是147分。

按照市面上的说法，高考英语147分这要背多少单词？一万，一万五？要用什么样的方法来背？什么词源法，什么词根词缀法，还是什么北大A4纸法？

然而通通都不是。

今天我们将公开我们这里背单词的秘密：那就是不背单词。

膨胀螺栓自从加入我们了以后，就不再被土妹允许背单词。是的，你没有听错，是不允许背单词，而是去做真正的积累单词。

2.你会觉得我们标新立异​

但问题不在我们身上，而是在于为什么你会觉得学英语就应该去背单词？

学习一门语言，当然需要词汇。然而，为什么是背单词？

在今天的中国，学英语就要背单词似乎已经成了一种共识，无论是出国的雅思托福，还是国内的考研，高考，中考，都特别强调背单词。无论是学校老师，还是外面培训机构，还是学霸分享，都无一例外地强调背单词。

以至于不背单词的我们反而成了特例，反而成了怪胎……

难道背单词就是积累词汇最好的方式？

为什么中国人学汉语，没有背过汉字3500？为什么美国孩子学英语，没有背过英语3500？为什么到了现在中国孩子学英语，就有了初中1800，高中3500？

这非常非常地奇怪！

作为孩子的父母，你当年学英语的时候，是靠这样背单词来学英语的吗？

给大家讲个故事：

某机构做出国留学的时候，很多同学拿到国外大学的条件offer了，急需快速通过考试。条件性offer就是你其他要求都达到了，只需要雅思托福达到了指定的分数，就可以去留学了。

时间真的很紧，实力差得远，怎么办？慢慢积累词汇积累实力已经来不及了，怎么办？

反正也没有别的方法，死马当成活马医，那就背单词吧。只要来的学生足够多，总有一些学生通过背单词，考过了雅思托福考试，拿到了offer，风风光光地去国外留学了……

背单词确实就有备考效率高的优点。

至于背单词的缺点吧，学生却看不见……

那些没有通过背单词通过考试的人，他也不好意思说这里的方法无效，只好埋怨自己毅力不够，努力不够……

于是背单词这种原本只在应急时候才使用的，对部分人才有效的方法，具有了无法证伪的特点。

培训机构的其他业务线也大力推广背单词，四六级，高中，初中，也开始背单词……

这样的做法快速得到了扩散：

其他培训机构也效仿，学校老师也效仿。因为这是一个摆脱自己责任的方法。你没背下来，那就是你的事，那就是你的毅力问题，不是老师的事。

然而问题的根源却是为什么你会喜欢背单词——因为你想要快速改变​

首先，你可能从来没有用过英语，英语对你而言，完全只是应付考试，是考过即弃的负担。

真正用得好英语的家长，是不会让孩子这样去背单词的。

其次，最重要的是，你太想要一剂猛药，你想要马上就考好，马上就追上别人，甚至超过别人。你没有耐心慢慢积累……

最后，知你想要，投其所好。因为你想要快速改变……于是乎，遍地都是从一个月背完3500，到5天背完10000个单词，什么规律法，词源学，A4纸背单词法，什么词根词缀，什么联想记忆，什么样的妖魔鬼怪，什么样的美女画皮，都来了……

而且都是高赞……哪个平台都一样……

然而去的那么多人，有没有人达到目标？肯定有。有多少？10%，还是1%，不确定，反正很少……

谁要是真的搞出一个一般人5天就能背完10000个单词的方法，这方法太有用了，国家科技进步一等奖他至少要该拿吧？中科院院士该他当吧？新闻联播该天天上了吧？还在这里吆喝？

然而，当你想要，又所有人都这样说的时候，你就信了。​

当你的孩子真的去尝试时，他可能并没有成为那少数的幸运者。别人说5天就搞定的10000个单词的事情，他5年，10年可能都搞不定。

于是，不仅你自己责怪孩子，孩子自己也责怪自己，怪他不努力，怪他没恒心，怪他记忆力差……

作为家长你很着急，但你看不见房间里的大象，你却从来没有怀疑过背单词的问题。

3.背单词还带来了一系列恶果……​

缺乏深度理解：导致学生只记住单词的表面含义，而忽视了单词在不同语境中的含义和用法。

缺乏应用能力：没有在实际语境中使用，不知道如何正确地使用这些词汇。

记忆效果不持久：缺乏实际应用，机械记忆的词汇很可能会被快速遗忘。

降低学习兴趣：对于许多学生来说，机械记忆会变得枯燥乏味，降低他们的学习兴趣和动力。

因此，明眼人都知道，靠背单词的孩子，注定走不远……你会奇怪为什么以前孩子学得好好的，现在学不懂了呢？

至于你的孩子走不远，那不是初中老师的事情。你通过初中背单词，到了高中不行，但是你的初中升学率上去了，只能怪你高中跟不上……

当然，也不是没有例外，只要人多了，就总会有例外。有的人就是毅力极强，有的人就是学习能力极强，不仅能通过眼下的考试，在后面也能通过补救，让自己的水平跟上来……但这只是少数。

因此，你的孩子更可能成为这幸运的少数，还是那不幸的大多数？

如果你只剩下最后一个月就要高考，中考了，那背单词也无可厚非。

如果时间还早，那为什么一定要选择这种方式？

如果你的孩子才高三上学期，才高一上学期，甚至才初三上学期，初一上学期。你并不是立刻就需要一个分数成绩。

你为什么就要通过磨灭学生的兴趣，然后再要求他坚持？

你为什么要让他经历反复的挫败来考验他的意志？

你为什么只让他学应试英语，而不是积攒自己的实力？

难道让他学好了真英语，学到了会用的英语，他对英语会没有兴趣吗？

难道让他积攒了实力，他会做不好考试的试卷吗？

难道英语对他而言只是用来应付考试的吗？

4.该如何积累单词？​

幸运的是，背单词不是唯一的方式，也不是最好的方式。

最有效的词汇积累方法，就是用单词，包括阅读、写作、对话等，在使用单词的过程中积累单词，不仅能记住新的词汇，更重要的是还能理解和正确使用它们。

这里尤其强调阅读。只要几本适合自己当下水平的英语书就可以了。一篇文章有95%的单词孩子都不认识，自然不能读下去。然而，一篇文章只有5%的单词不认识，孩子不仅可以读下去，还会享受阅读的乐趣，还能开阔自己的视野，你还能有效地积累这5%的新单词。

用适合自己难度的文章，用我们滚雪球的方法，反复阅读，孩子的词汇量就会大量提升，孩子的词汇运用能力就会显著提升，孩子的文章理解能力就会显著提升。

没了，就这么简单。

长久。这样积攒实力的方法，注定你会越学越轻松，当别人卷不动的时候，你会成为别人眼中的学霸。

运用。你学会的不仅仅是考试考高分的英语，还是能够用起来的英语。

最大的优点是可以走得更远

一开始用这样的方法积累单词，你可能会觉得有点慢，但是这避开了背单词的所有缺点。

一开始虽然慢一点，但是可以很稳，而且还能走得很远……

当膨胀螺栓这个同学按照滚雪球的方法做了一学期以后，她就已经吊打了99%的高中英语老师了。

你见过几个中学英语老师一年读完了10本英语原著的。

高三下学期后，他就没有下过145分。高考英语147分，不是什么逆袭，也含有什么运气，这就是实力。

这就是“用”单词，和背单词的本质差异。

然而，就算你从理智上知道了我说的都是真的，你会有勇气从长远出发吗？

铺天盖地的说法，可是要让孩子背单词。还是说，你就只是想要，孩子的分数立刻就改变？

如果你认为积累实力太慢了，你只是想要把最近的考试分数弄高了就行……

那让你的孩子继续背单词吧……

毕竟谁也无法叫醒一个装睡的人……

如果你想要孩子真的学好英语，如果你想要孩子可以走得更远，你可以和我一起，从当下开始积累，一年以后，还一个不一样的孩子给你……
