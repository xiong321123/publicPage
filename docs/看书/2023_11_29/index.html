          • 日期和时间

            • 时间戳，以一个时间为点，和这个时间点的相对值，因为时间是固定的，一维的，所以理解起来相对简单
            • 日期和时间Date()是一个内建对象。很多都是对象。面向对象的编程语言。

          • JSON 方法，toJSON

            • json是为了解决对象转为字符串的传送，并将字符串转化为对象的问题而产生的。
            • Json：可以方便地将对象转化为Json字符串并将Json字符串转为对象
                • JSON.stringify 将对象转换为 JSON。 -JSON.parse 将 JSON 转换回对象。
            • 对象以外的其他原始值数据，也可以转为JSON字符串
            • JSON支持嵌套的对象转换，但是不支持循环引用。
