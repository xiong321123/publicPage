          • 在swift中的switch语句单个case可以复合匹配多个值，比如case "a", "A"。在JS中也可以这样用单个case进行复合匹配多个值吗？
          • 在 JavaScript 中，switch 语句不支持直接使用逗号分隔的方式来匹配多个值，类似于 Swift 中的 case "a", "A"。在 JavaScript 中，switch 语句通常是按照单个值进行匹配的。
          • 然而，在 JavaScript 中可以使用多个 case 语句来实现类似的效果，即在一个 case 分支中处理多个值。这可以通过在一个 case 下使用多个相同的处理语句来完成。
          • 例如：
          • var letter = 'a';
            - switch (letter) {
            case 'a':
            case 'A':
              console.log('Letter is a or A');
              break;
            case 'b':
              console.log('Letter is b');
              break;
            default:
              console.log('Letter is not a, A, or b');
            }
          • 在这个例子中，当 letter 的值是 'a' 或者 'A' 时，case 'a': 和 case 'A': 下面的代码都会被执行。这种方法可以模拟在单个 case 下匹配多个值的效果。
          • 这一点要注意，容易搞错，且不知道原因在哪里
          • var申明的变量没有作用域，是全局变量。而且在申明前就可以使用。所以尽量使用let来申明变量。 background-color:: yellow
          • const申明常量以后不能再次直接赋值，但是可以通过下标的形式修改
          • 对象用方括号访问，比点记法多了两点：可以访问属性名为多个字符串之间有空格的，以及可以在方括号中使用复杂的 表达式，比如字符串插值，甚至字符串拼接。
