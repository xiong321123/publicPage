2023_12_26

  • 昨天天然气恢复了，不用去镇上了，而且昨天晚上土豆没有发烧了。
  • 最近老婆的心情好多了。
  • [[笑来]][[专注的真相]]14课冥想可能不是最好的方式
    • 冥想是有效的，经常被推荐，但也经常被一些搞灵修的用来骗钱。
    • 其他和冥想一样有效果的方式，包括如下：
    • 体育运动，如跑步。
    • 阅读，调用多个器官在一个事情上，让自己更加专注。提高注意力跨度，本身阅读还有更多的其他好处。
    • 笑来的弹唱，调用更多器官，让自己专心致志的时间更长，提高注意力跨度。——从音乐上来说，笑来的水平实在太低了，但是这对笑来的益智和护智，非常非常的重要。
    • 益智：做这样的事情能让我们非常放松，非常专注，就相当于大脑在做有氧运动，大脑在撸铁，这非常助于提高注意力专注程度，注意力广度。
    • 对于我们其他人一样，有自己的一个爱好，而这种爱好能主动地让我们调用更多的器官，专注于一件事情上，哪怕我们做这个事情的水平依然非常低下，这样的爱好也是非常重要的。这个爱好就可以给我们带来快乐，提升我们的智力，保护我们的智力，保护我们不被其他空虚占据，不被游戏，短视频等内容轻松吸引走，有非常大的价值。
    • 其他的文体活动，艺体活动，都是这样的，都是好处巨大。但现在很多家长（也包括我自己）太过于功利化地看待艺体活动，仅仅认为有助于升学的才重要，仅仅认为能拿奖，能拿名次，能帮助升学，能工作挣钱，能换来饭吃的才是重要的。而这样的做法简直太狭隘了。
    • 2024年的一个目标，我们家人运动，还要有自己的爱好，土豆的爱好是画打仗的话，老婆的爱好是尤克里里，我的爱好是？或许我的爱好是做东西，比如折纸船。
    • 没做好这件事情有什么坏处：没有好的方式来放松，自己不够放松，对家人的态度就会有不好的情绪，家庭关系恶化，和家人吵架，老婆不开心，甚至自我伤害。孩子在一个不好的环境下生活，自己也非常难过，低沉。也做不好事情，也不可能有好的成长。因为没有这样的内容，我们就更容易被不好的游戏，短视频，社交网络给吸引走，比如孩子沉迷于手机和游戏，孩子接触于一些不好的小孩成为朋友。自己需要去其他地方寻找认同。
    • 做好了这件事情的好处：我们有了一个放松的方式，我们明白了生活中的意义，找到了自我平静，自我修复，自我愉悦的方式方法。我的智力能得到提升，专注能力能得到提升，能更好地学习新东西，也才能更好地用于工作，生产，挣钱。我们的家庭会有更好的建设。我们家也可以像穆兰的娘家一样，有自己的春节联欢晚会。我们会有更好的情趣。
  • [[笑来]][[专注的真相]]15那些被忽略的最该专注的重要事项
    • 受到屏幕影响的，不仅仅是工作和学习，还有更重要的地方，睡眠，运动，放松，学习，工作，其他……
    • 这里面的顺序很重要。在这之前，我忽略了这三点的重要性。
    • 睡眠，运动，放松，对于我们的大脑非常非常的重要的。需要刻意的专注。
    • 这确实是超出想象的，没想到这三样这么重要。以前看到的书籍，以前在学校里听说的学霸，在知乎上看到的文章，总是看到作者牛人是每天4,5点起床的，牛人总是在努力的工作。看到这些，都非常让我自责，觉得自己太不够努力了，我应该少睡觉，我应该把努力的这根弦绷得更紧一些。就想要减少睡眠时间，减少放松，减少运动。甚至这也影响了老婆，让老婆没有这样的放松，让老婆紧张了。看到笑来这样说，真的太放松了，终于可以好好休息而不自责了。另外，以后土豆上中学了，也可以这样，不要让他被卷，不要让他用错误的方式去学习。
    • 类似于手机需要电池和电量，人的大脑超级耗能耗电，神经元之间的沟通靠放电。而很多人的大脑是处于电量不足之中而不自知。大脑的电池，每个人是有显著差异的，有人只有2500毫安，有人则有一万多毫安，这是真实的差距。看不见，却真实存在的差异。人们对自己大脑却没有电量焦虑。
    • 睡眠，相当于电池充电。睡眠不好的情况下，学习，工作都效率太低。睡眠不佳还会感受到更大的压力和焦虑。
      • 导致睡眠不好的方式，睡觉前一小时接触到屏幕，导致深度睡眠时间降低了1/4，又因为沾上了手机，睡眠时间靠后，从而导致睡眠时间缩短。还有的人，带着手机上床。
      • 对策，睡觉前一小时坚决不接触手机，并把手机放在离床比较远的地方。我们晚上关上电脑了以后，做做家务，和老婆聊天。
    • 运动，相当于电池维护，扩容。以前的人不需要刻意运动，因为工作生活中的体力活动太多，身体得到了足够的锻炼。现在不一样了，工作中没有那么多锻炼我们的机会，我们的身体整体而言变得更差了。
      • 不要等到身体坏掉了，得了大病，吃了大亏，才开始重视健康，才开始锻炼身体。那样的做法通常只会维持一个非常短暂的时间，等到大病好了一段时间，就又忘了。我的经历确实就是这样的。之前在大学城医院住院。
      • 我们需要把运动放到一个非常高的高度，而在运动之前，我们也需要先去学习，买几本书，报一下课，请个健身教练，都是非常划算的。健身类的东西被市场低估了，价格低于其价值，所以去购买就非常划算。在日常中，更应该是每天都运动一小时左右。土豆打乒乓球也可以买书买课，看视频。
    • 放松，相当于电池优化。放松也被屏幕给污染了。躺在那里玩手机，看电视，打电子游戏，并不是好的放松，虽然没有体力运动，但是依然消耗了脑力，消耗了大脑的电池电量。这样的放松，反而让人更累。这就导致大脑电量越来越低
      • 现在知道了，放松就是去做自己爱好的事情，放松大脑还可以是泡澡，散步，发呆，晒太阳，唱歌，跳舞，老婆还喜欢和人聊天，疯架，斗鸡，温泉，床上聊天，睡觉，尤克里里，土豆的画画，我的做东西，做纸船，抓鱼，采摘，光流，去西南大学，去歌乐山，九凤瑶池，去渝西的一些景点，青龙湖，长寿湖的浆板……做这些事情的时候要专注，不能同时进行多任务。
    • 在大脑有个好电池以后，才是学习和工作。也才能学习好和工作好。
    • 以学习的方式来工作，学习放在前面，学习第一，从工作中努力学习。
    • 为自己工作，在任何时候都为自己工作，打工时也为自己工作，可以减少内耗。彻头彻尾为自己打工，还有一个非常大的好处，就是可以睡到自然醒，这是非常大的优点。
    • 原来自己就是最好的智能设备，对自己的身体好一点，是这样的方式。原来，对自己好一点，是这样的重要，而不是像以前以为的那样，让自己更努力，让自己更辛苦……
    • 原来，在这个时代，让自己活得久一点，让自己的状态好一点，才是真正的优势策略，才可以做更多的事情。好放松呀，原来这才是对的，自己以前被灌输的很多内容，都是错误的。
    • 晚上我和老婆讨论了这个事情，老婆初中，大学的时候，很多的睡觉，休息，放松，成绩反而非常好。高中的时候，以及和我在一起了以后，就绷得更紧一些，状态就很不好，甚至也不知道挣钱存到今后有什么用处，就被我带偏了。我觉得我就是被主旋律的给带偏了，我们都是被告知，谁谁谁4点5点就起床起来学习工作，阅读了，谁谁谁工作到多久才睡觉。我就感觉我很惭愧，很想要自己努力起来，否则觉得自己太浪费光阴了。把自己绷得太紧了。
    • 为什么主旋律没有重视放松：可能是以前的工作，不需要多少的脑力，不需要多少的创新，之前的工作产出和时间更相关，在效率都相差无几的情况下，谁的工作时间更长，最终谁就更有产出。谁要是提倡放松，反而被认为是不务正业，不上进。但是现在不一样了，而现在的工作，已经不是和以前的工作那样了，是需要更多脑力，效率差异更大了，到了最后甚至出现了指数级的差异。最终大家在效率上的差异是10倍，不是多工作，少放松的那5%的时间。
    • 没做好这件事情的后果：生病，甚至可能是大病，用光我们多年积攒下来的那一点钱，因病返贫，也包括死亡以及寿命大幅度缩短，或者是今后的高质量的生命大幅度缩短。还包括孩子与我们大人的心理疾病。比如产后抑郁，孩子抑郁，找不到生命的意义，自残。这些都是现实的，不是夸张的。吵架，离婚，更是非常的可能。还有就是幸福度极低，非常努力地苦兮兮的努力，结果却并不好。永远也不可能自由，不可能时间自由，财务自由
    • 做好了这件事的好处：我们的生活更加幸福，大家都更加地开心，也才能更好地爱着对方。然后也才能更好地锻炼我们的大脑，也才能不被手机轻易吸引走。寿命更长，做更多的事情。睡眠，运动，放松，都是为了能更好地做好事情，做更多的事情，这才是努力所追求的目标。
