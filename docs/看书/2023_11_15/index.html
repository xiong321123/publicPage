              • 插入元素：​

                使用 splice() 方法可以在数组的指定位置插入一个或多个元素，同时也可以删除元素（如果指定了删除的数量）。

                示例：在数组中间位置插入元素

                let arr = [1, 2, 4, 5];
                // 在索引为 2 的位置插入元素 3
                arr.splice(2, 0, 3);
                console.log(arr); // 输出: [1, 2, 3, 4, 5]

              • 删除元素：​

                同样地，splice() 方法也可以删除数组中的一个或多个元素。

                示例：删除数组中间位置的元素

                let arr = [1, 2, 3, 4, 5];
                // 从索引为 2 的位置开始删除 1 个元素
                arr.splice(2, 1);
                console.log(arr); // 输出: [1, 2, 4, 5]

                splice() 方法的第一个参数是起始索引，第二个参数是要删除的元素数量（如果是 0，则表示不删除元素），接着可以在第三个参数开始的位置插入新元素。

                通过 splice() 方法，你可以在数组的中间位置动态地添加或删除元素。

            • 关于 “length”​

            • 当我们修改数组的时候，length 属性会自动更新。准确来说，它实际上不是数组里元素的个数，而是最大的数字索引值加一。
            • length 属性的另一个有意思的点是它是可写的。
            • 如果我们手动增加它，则不会发生任何有趣的事儿。但是如果我们减少它，数组就会被截断。该过程是不可逆的，下面是例子：
            • let arr = [1, 2, 3, 4, 5];
              - arr.length = 2; // 截断到只剩 2 个元素
              alert( arr ); // [1, 2]
              - arr.length = 5; // 又把 length 加回来
              alert( arr[3] ); // undefined：被截断的那些数值并没有回来
            • 所以，清空数组最简单的方法就是：arr.length = 0;。

            • 总结​

            • 数组是一种特殊的对象，适用于存储和管理有序的数据项。
            • 声明：
            • // 方括号 (常见用法)
              let arr = [item1, item2...];
              - // new Array (极其少见)
              let arr = new Array(item1, item2...);
            • 调用 new Array(number) 会创建一个给定长度的数组，但不含有任何项。
            • length 属性是数组的长度，准确地说，它是数组最后一个数字索引值加一。它由数组方法自动调整。
            • 如果我们手动缩短 length，那么数组就会被截断。
            • 获取元素：
            • 你可以通过元素的索引获取元素，例如 arr[0]
            • 我们也可以使用允许负索引的 at(i) 方法。对于负值的 i，它会从数组的末尾往回数。如果 i >= 0，它的工作方式与 arr[i] 相同。
            • 我们可以通过下列操作以双端队列的方式使用数组：
            • push(...items) 在末端添加 items 项。
            • pop() 从末端移除并返回该元素。
            • shift() 从首端移除并返回该元素。
            • unshift(...items) 从首端添加 items 项。
            • 遍历数组的元素：
            • for (let i=0; i<arr.length; i++) — 运行得最快，可兼容旧版本浏览器。
            • for (let item of arr) — 现代语法，只能访问 items。
            • for (let i in arr) — 永远不要用这个。
            • 比较数组时，不要使用 == 运算符（当然也不要使用 > 和 < 等运算符），因为它们不会对数组进行特殊处理。它们通常会像处理任意对象那样处理数组，这通常不是我们想要的。
            • 但是，我们可以使用 for..of 循环来逐项比较数组。
            • function compareNumeric``(``a, b``) {if (a > b) return 1``;if (a == b) return 0``;if (a < b) return -``1``;}
            • let arr = [ 1``, 2``, 15 ]``;
            • arr.``sort``(compareNumeric)``;
            • alert``(arr)``;
            • 上面这段代码为什么是按照升序排列的没有搞懂 background-color:: yellow
            • 对于Array.sort()方法，当提供一个比较函数时，该函数应该返回三种可能的值：
            • 若返回值小于 0，则 a 应该排在 b 前面。
            • 若返回值等于 0，则 a 与 b 位置保持不变。
            • 若返回值大于 0，则 b 应该排在 a 前面。
            • 在你提供的 compareNumeric 函数中，当 a > b 时返回 1，当 a == b 时返回 0，当 a < b 时返回 -1。这样的设定是为了符合升序排序的要求。这个比较函数告诉 sort() 方法：如果 a 大于 b，则将 a 排在 b 之后，如果 a 小于 b，则将 a 排在 b 之前。因此，数组将按升序排列。
            • 如果想要进行降序排序，比较函数需要相应地进行调整，确保当 a > b 时返回 -1，当 a == b 时返回 0，当 a < b 时返回 1。这样 sort() 方法就会根据比较函数返回值的不同，正确地对数组进行降序排序。

            • 总结​

            • 数组方法备忘单：
            • 数组的一些方法在swift中有接触过
            • 添加/删除元素：
            • push(...items) —— 向尾端添加元素， collapsed:: true
              • pop() —— 从尾端提取一个元素，
              • shift() —— 从首端提取一个元素，
              • unshift(...items) —— 向首端添加元素，
              • splice(pos, deleteCount, ...items) —— 从 pos 开始删除 deleteCount 个元素，并插入 items。
              • slice(start, end) —— 创建一个新数组，将从索引 start 到索引 end（但不包括 end）的元素复制进去。
              • concat(...items) —— 返回一个新数组：复制当前数组的所有元素，并向其中添加 items。如果 items 中的任意一项是一个数组，那么就取其元素。
            • 搜索元素：
            • indexOf/lastIndexOf(item, pos) —— 从索引 pos 开始搜索 item，搜索到则返回该项的索引，否则返回 -1。 collapsed:: true
              • includes(value) —— 如果数组有 value，则返回 true，否则返回 false。
              • find/filter(func) —— 通过 func 过滤元素，返回使 func 返回 true 的第一个值/所有值。
              • findIndex 和 find 类似，但返回索引而不是值。
            • 遍历元素：
            • forEach(func) —— 对每个元素都调用 func，不返回任何内容。
            • 转换数组：
            • map(func) —— 根据对每个元素调用 func 的结果创建一个新数组。 collapsed:: true
              • sort(func) —— 对数组进行原位（in-place）排序，然后返回它。
              • reverse() —— 原位（in-place）反转数组，然后返回它。
              • split/join —— 将字符串转换为数组并返回。
              • reduce/reduceRight(func, initial) —— 通过对每个元素调用 func 计算数组上的单个值，并在调用之间传递中间结果。
            • 其他：
            • Array.isArray(value) 检查 value 是否是一个数组，如果是则返回 true，否则返回 false。
            • 请注意，sort，reverse 和 splice 方法修改的是数组本身。
            • 这些是最常用的方法，它们覆盖 99％ 的用例。但是还有其他几个：
            • arr.some(fn)/arr.every(fn) 检查数组。
            • 与 map 类似，对数组的每个元素调用函数 fn。如果任何/所有结果为 true，则返回 true，否则返回 false。
            • 这两个方法的行为类似于 || 和 && 运算符：如果 fn 返回一个真值，arr.some() 立即返回 true 并停止迭代其余数组项；如果 fn 返回一个假值，arr.every() 立即返回 false 并停止对其余数组项的迭代。
            • 我们可以使用 every 来比较数组：
            • function arraysEqual(arr1, arr2) {
              return arr1.length === arr2.length && arr1.every((value, index) => value === arr2[index]);
              }
              - alert( arraysEqual([1, 2], [1, 2])); // true
            • arr.fill(value, start, end) —— 从索引 start 到 end，用重复的 value 填充数组。
            • arr.copyWithin(target, start, end) —— 将从位置 start 到 end 的所有元素复制到 自身 的 target 位置（覆盖现有元素）。
            • arr.flat(depth)/arr.flatMap(fn) 从多维数组创建一个新的扁平数组。
            • Array.of(element0[, element1[, …[, elementN]]]) 基于可变数量的参数创建一个新的 Array 实例，而不需要考虑参数的数量或类型。

              • 在 JavaScript 中，forEach，for，for..of 和 for...in 是用于遍历数组或对象的不同方法，它们之间有一些区别： background-color:: yellow collapsed:: true

                1. forEach 方法：

                • forEach 是数组的原生方法，用于遍历数组中的每个元素。

                • 它接受一个回调函数作为参数，在数组的每个元素上执行该回调函数。

                • forEach 不返回新的数组，只是对现有数组进行迭代操作，无法在迭代中使用 break 来提前结束循环。

                  const array = [1, 2, 3];
                  array.forEach(element => {
                  console.log(element);
                  });
                  1. for 循环：

                • for 循环是常见的迭代方式，可以用于遍历数组和对象。

                • 通过设置初始条件、循环条件和递增表达式来控制循环。

                • 可以使用 break 和 continue 来控制循环的执行。

                  const array = [1, 2, 3];
                  for (let i = 0; i < array.length; i++) {
                  console.log(array[i]);
                  }
                  1. for..of 循环：

                • for..of 是 ES6 引入的用于遍历可迭代对象（如数组、字符串、Map、Set 等）的语法。

                • 可以直接迭代数组的值，而不需要索引，提供了更简洁的语法。

                • 不适用于普通对象，因为普通对象并非可迭代对象。

                  const array = [1, 2, 3];
                  for (const element of array) {
                  console.log(element);
                  }
                  1. for...in 循环：

                • for...in 用于遍历对象的可枚举属性（包括继承得到的属性）。

                • 不推荐用于遍历数组，因为它遍历的是数组的索引，可能会包含非数字类型的键名以及原型链上的属性。

                  const object = { a: 1, b: 2, c: 3 };
                  for (const key in object) {
                  console.log(key + ': ' + object[key]);
                  }

                  总结：

              • forEach 用于数组，用于遍历数组元素，并在每个元素上执行回调。

              • for 和 for..of 用于遍历数组，其中 for 更加灵活，可以通过索引控制，而 for..of 提供了更简洁的语法。

              • for...in 用于遍历对象的属性，不推荐用于遍历数组。
