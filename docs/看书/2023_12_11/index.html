                • 有一种特殊的语法，称为“前瞻断言（lookahead）”和“后瞻断言（lookbehind）”。
                • 首先，让我们从字符串中查找价格，例如 1 turkey costs 30€。即：一个数字，后跟€符号。
                • 前瞻断言 语法为：x(?=y)，它表示“仅在后面是 Y 时匹配 X”。这里的 X 和 Y 可以是任何模式。
                • 那么对于一个后面跟着 € 的整数，正则表达式应该为：\d+(?=€)。
                • let str = "1 turkey costs 30€";

              • 正向先行断言（positive lookahead）和负向先行断言（negative lookahead）是正则表达式中的两种先行断言。它们用于在匹配模式的同时，检查模式之后的内容是否符合特定条件。

              • 正向先行断言使用 X(?=Y) 的语法，表示“查找 X，但只有在后面跟着 Y 时才匹配”。这里的 X 和 Y 可以是任何模式。例如，要查找紧跟在数字后面的欧元符号（€），可以使用 \d+(?=€) 的正则表达式。

              • 负向先行断言使用 X(?!Y) 的语法，表示“查找 X，但只有在后面不跟着 Y 时才匹配”。比如，要查找不紧跟在数字后面的欧元符号（€），可以使用 \d+\b(?!€) 的正则表达式。

              • 这两种先行断言允许我们在匹配模式的同时，对模式之后的内容进行条件检查，从而更精确地匹配我们需要的内容。

              • 在正则表达式中，先行断言（Lookaheads）和后行断言（Lookbehinds）是用于检查匹配位置前后内容的特殊结构。它们允许你定义条件，用来确认在特定位置之前或之后的文本是否匹配某种模式。

              • 1. 先行断言（Lookaheads）​

              • a. 正向先行断言（Positive Lookahead）​

                正向先行断言用于确认某些文本后面的内容匹配某种模式，但不包含这些内容。其语法是 (?=...)。

                例如，使用正向先行断言匹配后面紧跟着数字的单词 "apple"：

                apple(?=\d)

                这个表达式将匹配 "apple"，但仅当其后跟着一个数字时。

              • b. 负向先行断言（Negative Lookahead）​

                负向先行断言用于确认某些文本后面的内容不匹配某种模式。其语法是 (?!...)。

                例如，使用负向先行断言匹配后面不是数字的单词 "apple"：

                apple(?! \d)

                这个表达式将匹配 "apple"，但仅当其后不跟着一个数字时。

              • 2. 后行断言（Lookbehinds）​

              • a. 正向后行断言（Positive Lookbehind）​

                正向后行断言用于确认某些文本前面的内容匹配某种模式，但不包含这些内容。其语法是 (?<=...)。

                例如，使用正向后行断言匹配前面紧跟着数字的 "$" 符号：

                (?<=\d)\$

                这个表达式将匹配 "$"，但仅当其前面是一个数字时。

              • b. 负向后行断言（Negative Lookbehind）​

                负向后行断言用于确认某些文本前面的内容不匹配某种模式。其语法是 (?<!...)。

                例如，使用负向后行断言匹配前面不是数字的字母 "X"：

                (?<![0-9])X

                这个表达式将匹配 "X"，但仅当其前面不是一个数字时。

                这些断言结构允许你在进行匹配时对所在位置的内容进行条件检查，而不是直接匹配内容。它们是强大的工具，可以使正则表达式更加灵活和精确。需要注意的是，并非所有编程语言或正则表达式引擎都支持全部类型的先行断言和后行断言，因此在使用时需要查看相关的文档来确认支持的功能和语法。

                • 这个正则表达式 /(\w+) \1 \1/ 主要包含以下部分：
                  1. (\w+)：这是一个捕获组，用括号包裹起来，表示匹配至少一个或更多个字母、数字或下划线字符。\w 表示匹配字母、数字或下划线字符，+ 表示匹配前面的表达式一次或多次。捕获组被定义为第一个组（用 ( ) 括起来的部分）。
                  1. \1：这不是一个普通的字符匹配，而是一个反向引用，它引用了第一个捕获组 (\w+) 所匹配到的内容。\1 表示匹配前面第一个捕获组所匹配到的内容。在这个正则表达式中，\1 会尝试匹配和第一个捕获组 (\w+) 相同的内容。
                  1. 空格和重复的 \1：这表示两个空格之间重复匹配第一个捕获组所匹配的内容。
                • 所以这个正则表达式的含义是：它会匹配三个由空格分隔的相同的单词或字符序列。例如，如果字符串为 "hello hello hello"，则正则表达式会匹配整个字符串，因为它包含了三个连续重复的单词 "hello"，每个单词之间由空格分隔。
                • 但需要注意，这个正则表达式要求重复的内容在同一行内出现，并且每个重复的单词之间必须由一个空格分隔。

                • 除了 test() 方法外，JavaScript 的正则表达式还有一些其他常用的方法：
                • 这下面的exec(),match(),search(),有一些类似的地方，都有匹配这个基础功能。 background-color:: yellow
                    • exec() 是正则表达式对象的方法，返回单个匹配项的详细信息或 null。正则表达式的方法
                    • match() 是字符串对象的方法，返回所有匹配项的数组或者单个匹配项及其捕获组。字符串的方法
                    • search() 是字符串对象的方法，返回匹配的第一个字符位置索引或 -1。
                    • exec() 和 match() 返回匹配内容或者内容的详细信息，而 search() 返回匹配内容的位置索引。
                    • exec() 在具有全局标志 g 时，可以多次调用以查找所有匹配项，而 match() 会一次性返回所有匹配项的数组。
                  1. exec(): 这个方法在一个指定字符串中执行一个搜索匹配。如果找到了匹配，就返回一个结果数组，否则返回 null。返回的数组包含匹配的文本，以及捕获组（如果有的话）的信息。
                • 示例：
                  const regex = /hello/;
                  const str = 'hello world';
                  const result = regex.exec(str);
                  console.log(result); // 输出匹配的结果数组，例如 ["hello", index: 0, input: "hello world", groups: undefined]
                  1. match(): 这个方法用于在字符串中检索一个或多个与正则表达式匹配的文本。如果匹配成功，match() 方法会返回一个数组，其中包含匹配的结果；如果未找到匹配，则返回 null。
                • 示例：
                  const regex = /hello/;
                  const str = 'hello world';
                  const result = str.match(regex);
                  console.log(result); // 输出匹配的结果数组，例如 ["hello"]
                  1. search(): 这个方法用于检索与正则表达式相匹配的字符串。如果找到匹配，则返回匹配的起始位置；如果未找到匹配，则返回 -1。
                • 示例：
                  const regex = /hello/;
                  const str = 'hello world';
                  const result = str.search(regex);
                  console.log(result); // 输出匹配的起始位置，例如 0
                  1. replace(): 这个方法用于替换字符串中与正则表达式匹配的子串。它可以将匹配的文本替换为指定的字符串或者函数的返回值。
                • 可以通过将内容替换为‘’‘’空，来实现去除某些不想要的字符串 background-color:: yellow
                • 示例：
                  const regex = /hello/;
                  const str = 'hello world';
                  const newStr = str.replace(regex, 'hi');
                  console.log(newStr); // 输出替换后的字符串，例如 "hi world"
                  1. split(): 这个方法使用正则表达式或者字符串作为分隔符，将字符串拆分为数组。它根据匹配到的模式将字符串拆分为子字符串，并返回一个数组。
                • 示例：
                  const regex = /\s+/;
                  const str = 'hello world';
                  const result = str.split(regex);
                  console.log(result); // 输出拆分后的数组，例如 ["hello", "world"]
                  1. toString(): 这个方法返回正则表达式的字符串表示。可以用于将正则表达式转换为字符串。
                • 示例：
                  const regex = /hello/;
                  const regexString = regex.toString();
                  console.log(regexString); // 输出正则表达式的字符串形式，例如 "/hello/"
                  1. test()（RegExp实例的方法）: 这个方法用于检测字符串是否与正则表达式匹配，并返回 true 或 false。
                • 示例：
                  const regex = /hello/;
                  const str = 'hello world';
                  const result = regex.test(str);
                  console.log(result); // 输出 true
                • 这些方法可以帮助你在 JavaScript 中更有效地使用正则表达式进行字符串操作、匹配和替换。
                • 在 JavaScript 中，findAll() 方法和 sub() 方法并不是正则表达式对象上的原生方法。然而，在其他编程语言或特定的库中，比如 Python 的 re 模块，有类似的方法用于正则表达式操作。
                • 在 Python 的 re 模块中：
                  1. findall() 方法： 这个方法用于在给定字符串中查找所有与正则表达式匹配的非重叠模式，并返回匹配的结果作为一个列表。
                • 示例（Python）：
                  import re
                  - text = "Hello, my phone numbers are 12345 and 67890"
                  pattern = r'\d+'  # 匹配数字
                  matches = re.findall(pattern, text)
                  print(matches)  # 输出匹配到的所有数字列表，例如 ['12345', '67890']
                  1. sub() 方法： 这个方法用于在给定字符串中，使用指定的替换字符串替换所有与正则表达式匹配的模式。
                • 示例（Python）：
                  import re
                  - text = "Hello, my phone number is 12345"
                  pattern = r'\d+'  # 匹配数字
                  replacement = "*****"
                  new_text = re.sub(pattern, replacement, text)
                  print(new_text)  # 输出替换后的字符串，例如 "Hello, my phone number is *****"
                • 请注意，JavaScript 中并没有类似的 findAll() 或 sub() 方法。在 JavaScript 中，你可以使用的主要方法是 match()、exec()、replace()、search() 和 split()，它们能够实现类似的匹配、搜索、替换和拆分操作。

              • 正则表达式中的g：

                • 在 JavaScript 中，正则表达式中的 g 标志代表全局匹配模式（global）。当你在正则表达式末尾使用 g 标志时，它会指示正则表达式去查找输入字符串中的所有匹配项，而不是在找到第一个匹配项后就停止搜索。 background-color:: yellow
                • 在给定的正则表达式中 ^\s*|\s*$，^ 表示字符串的开始位置，\s* 表示匹配零个或多个空白字符（例如空格、制表符等），| 表示或的意思，\s*$ 表示匹配零个或多个空白字符，紧接着字符串的结束位置 $。
                • 结合 g 标志，这个正则表达式表示在输入字符串中匹配开头和结尾的空白字符，并移除它们。使用全局匹配模式 g，它会在整个字符串中查找所有符合条件的模式，并进行替换或其他操作。
                • 例如，在使用 replace() 方法时，结合 g 标志，可以移除字符串开头和结尾的空白字符：
                • let str = "  Hello, World!  ";
                  let wsRegex = /^\s*|\s*$/g;
                  let result = str.replace(wsRegex, ""); // 移除开头和结尾的空白字符
                  console.log(result); // 输出: "Hello, World!"
                • 在这个例子中，^\s*|\s*$ 匹配了字符串开头和结尾的空白字符，并通过 replace() 方法替换为空字符串，最终移除了开头和结尾的空白字符。
