  • 年底了，回顾一下这一年的经历吧：
    • 去年春节去了海南，我们计划要下一个小孩实施了，也成功怀孕。整个备孕期间，也都是在这一年里完成的。经过难熬的九个月，老婆在9.18的预产期，生下了玉米。生下玉米后，有月嫂田姐的帮助，我们在坐月子期间，轻松了很多。后来田姐走了，就我和老婆两人带娃，辛苦了许多。
    • 在海南这期间，我们推出了定制课，效果很好，学员得到了更好的服务，更好的提升，我们也收了比以前更多的钱。暑假再贵阳时，为了让生娃以后，我们的课程能继续运转下去，老婆在临产前努力的录制了很多新的课程。但是就在生下了玉米后不久，我们得知双减的加强版落地了，补课成了违法行为。经历了非常难熬的一段时间，我们决定放弃初高中的补课，然后转向雅思，这真的是非常艰难的时刻做出的重大决定。
    • 在这一年里，我加入了小白慢爬营，在这里面学习了一些编程的知识。这一年里，使用GPT也很多，编程的学习难度下降了很多。我也有较快的进步。不过总体而言，并没有达到努力。
    • 土豆通过学习playground开始学习了正式的变成语言，并在年底正式开始学习Python，结果令我和老婆吃惊，8岁小孩竟然可以理解蟒蛇书中的很多内容。
    • 在土豆的激励下，坐月子期间的老婆也开始学习编程，而且老婆学习编程，上手起来也比我之前的快得多。老婆真的非常励志。
    • 这一年里，老婆是非常难的，从怀孕初期就开始吐得很厉害，而且还开始了我们定制课的新模式，又在怀孕期间加班加点地出课，最后还受到了双减消息的打击，以及挣扎后决定放弃中高考，又在坐月子期间开始学习编程以及备课雅思。老婆备课雅思期间，让我见识到了一个猛人的操作。
    • 这一年里，土豆的变化也很大，学习能力进一步强化。读书更多，自己读完了一些书籍，比如富有的习惯等，目标是100本好书。
    • 这一年里，在笑来，溜达这里跟下来，收获也是非常大的，笑来的各种课程颠覆了我们的很多认知，我们家庭也会因此更好。
  • 新的一年里：
    • 要照顾好家人，家庭才是我们一切的基础。
    • 我要继续读书，继续学习笑来的课程，现在王渝林的日更，也给到了我很大的激励。

  • 土豆：
    • 土豆，昨天爸爸妈妈外出后，你帮外婆在工地上干了活，挣到了162元钱，你非常高兴。听到你说：靠自己挣到的钱，心里更踏实。我真为你感到高兴和骄傲。是的，拥有挣钱能力，是非常的重要的。今天早上你6点多就起床了，希望又去干活挣钱。看到你对挣钱，充满了渴望，爸爸更是高兴。只有更想要挣钱，最终更有可能挣到更多的钱。
    • 就像巴菲特，苏世民等人一样，他们也都是从小就对钱充满了渴望，并且从小就开始挣钱，从小就想尽办法挣钱。

    • 外婆说，让你去干活挣钱，是想要让你明白，学习的重要性。不过目前，似乎外婆的说法还不够清楚，需要我再向你解释一下。​

      • 在工地上挣钱，是用自己的体力和时间来换钱，但是在工地上挣钱的这类工作，不能挣到很多的钱。有两个原因，一个是我们可以出售体力的时间比较有限，一般也就四五十年。再一个，工地上这类体力劳动，收入增长比较有限。不会随着时间变化而呈现快速提高。
      • 想要挣更多的钱，一个是提升挣钱的速度，一个是提升挣钱的时间。
      • 不同人挣钱的速度很不一样，最大的差异就是学习导致的，当然，这里的学习不仅仅是在学校为了考试的学习，而是指所有的为了提升自己的能力的学习。
    • 1.收入增长很重要
    • 上图中横着的那根线是横坐标，代表时间，竖着的那根线是纵坐标，代表挣钱的速度。
    • 如果我们挣钱的速度不变，就像上图中那根水平的黄色的CD线。上图ABCD四个点框起来的面积，就代表我们能挣到的钱。如果我们不能不断地学习，甚至从学校毕业后就停止了学习，我们的收入曲线就会成为这种情况。这种情况虽然是是大多数人的情况，但不是一种好的状态。
    • 也有一些少数人通过不断地学习，挣钱的速度会越来越快，就如同上图中CG两点所在的那条曲线。上图中ABCG四个点框起来的面积，就代表在这种情况下我们能挣到的钱。
    • 你能轻易地看到，ABCG的面积远远大于ABCD的面积，也就是挣钱速度越来越快的情况下挣到的钱更多。
    • 每个人都想要成为挣钱速度越来越快的这种情况，但是如何才能实现呢？只有通过不断的学习才能实现。通过学习后才能发现社会的需求，才能生产出社会更加需要的产品和服务。再次重复，学习不仅仅是指学校课堂上的学习。
    • 除此以外，同样是收入越来越高的，也有差异。
    • !
    • 活得久模式
    • 比如一个人的收入是同样一根指数曲线，越来越高。一种情况下，他工作到B这个时间点就因为身体健康或者其他原因没有收入了。另一种情况是身体很健康，而且寿命比较长，他一直到F这个时间点都还有收入。有收入更久的这种情况下就可以多挣下BFED这四个点框起来的面积。而且由于他的收入是越来越高，所以他后面每多工作一年，就可以比之前的一年多收入多得多的钱。
    • 所以我们看到，身体健康很重要，活得久也很重要。这种重要的差异，会体现在人生的后端有显著的体现。
    • 早积累模式
    • 同样是一个人，他的收入遵循同一跟指数曲线，除了工作截止时间不同有差异以外，开始积累得早晚也有差异。比如这个人如果不是从A这个时间点开始积累，而是从M这个点就开始积累，同样到B这个时间点终止。从M这个时间点开始积累会比从A这个时间多出来FCDE这四个点框选出来的面积。早积累的效果和活得久的效果是一样的惊人。
    • 现实生活中，最终也有像巴菲特这样的神人，真的就做到了收入越来越高，遵循复利曲线的指数型增长。还做了活得久，现在他都93岁了。还做到了积累得早，巴菲特13岁就开始投资股票了。所以巴菲特被封神为股神，是实至名归。
    • 当然，生活中我们不仅要关注收入，还要消费。
      • 常见的有下面三种模式：
      • 为钱所困模式：
      • 常见的是这种模式，收入是CD这跟线，一直没有多大的增长，在工作期间消费是EF这根线。你能看到，由于消费的速度只是稍微低于收入的速度，这样下来，在工作期间存下来的钱就是CDEF框起来的面积，根本存不下来什么钱。更关键的是，人生还总有工作以前和结束工作以后，这两个时间段内只有消费没有收入。我们大部分人都是这种情况，一直被钱的各种问题所左右。
      • 高收入高消费模式：
      • 少部分人是这另外一种模式，这种模式下收入曲线CG越来越高，但于此同时，消费曲线EF也越来越高。最终，在工作期间存的钱钱，就是这个图中CGEF四个点框的那一部分，面积也不大，也没有存下来什么钱。这就是收入高但并不富裕的模式。这种情况下即使收入较高，但是依然没有自由，依然为钱所困。
      • 很多人收入提高了，就提高了消费，比如去买豪车豪宅，甚至还贷款去买，就很少感受到自由。
      • 真正的富裕模式是这一种，虽然收入EF这条线越来越高，但是消费依然维持在CD这个比较低的线，这样就能在工作期间存下来CDEF四个点框起来的区域，你能看到这个面积就比之前的大得多。这就是真正的富裕，有用不着花的钱，也就是有花不完的钱。这种情况下，人就能真正做到自由，不用为钱担心。
      • 这样的人虽然比较少，在我们身边也有，比如呆哥就是这样的。
      • 有剩下花不完的钱，才是富裕
    • 再来回顾一下重点：
    • 通过不断的学习，可以让我们今后挣钱的速度越来越快。通过早点进行必要的学习积累，我们在相同寿命下能挣到更多得不成比例的钱。通过锻炼身体，延长寿命，延长工作时间，我们也可以挣到多得不成比例的钱.
    • 这也是外婆希望通过这个活动，让你认识到学习的重要性。
    • 再一个，收入增加以后，如果消费也相应地水涨船高，其实也很难获得自由的。只有收入增加了后，依然让消费维持在比较低的水准，有用不着花的钱，有花不完的钱，才是真正的富裕。
    • 你也可以问一下外公外婆，为什么他们比其他工人能挣到更多的钱，他们是如何做到的。
