为什么孩子应该学编程

1.学习编程的好处​

一直都知道编程的重要性的，毕竟在很多书里面都看到过，在很多的课程也都听到过。至于接触到过多少次，我自己都记不清楚了。

长期的好处​

自从有国家以来，人类经历了农业时代，工业时代，正在步入信息时代。

在农业时代，农业生产的财富占据了整个社会财富的绝大多数。进入工业时代以后，由于效率的提升，农业生产的财富更多了，但农业生产占整个社会财富的比例却大幅度下降了，让位于工业所生产的财富。

以发生在我身边的事情为例，我的爸妈都是优秀的农民，农艺精湛，吃苦耐劳，体力超群。在我小时候，我家的地越种越宽，粮食牲畜越来越多，卖出去的钱也越来越多。但是在这个过程中，我并没有感受到我家更富裕了。相反，我感受到的却是我家的经济相对变差了。因为别人家放弃了庄稼地，选择了出门打工。我的家庭还在从事传统的农业生产，而其他的家庭却通过打工参与到了工业生产，也获得了更多的经济回报。

现在进入信息时代，农业和工业的效率也进一步提升，生产的财富总量也进一步增加。但农业和工业生产的财富占整个社会财富的比例却大幅下降了，让位于数字产业所生产的财富。

同样在我身边，也有看得见的例子。为了不从事农业生产，我大学选择了车辆工程这样的工科学科，希望参与工业生产。我的大学同学大多数现在都在汽车制造业工作，参与到了工业生产中。虽然大家每年的工资都有一些增长，但是都明显低于IT行业工资增长幅度。这样的情况和我父母当时务农，别的家庭选择了打工一样。

因为时代的发展，农业不在需要那么多劳动力了，我们这一代也就很少有人再回老家务农了。随着时代的继续发展，工业也不在需要如此多的劳动力，机器可以代替人更好地工作。

今后孩子们长大了不懂编程，别说在信息时代组织生产当老板了，就连参与到其中去打工，都找不到地方。

再一个，程序语言，本身就是面向机器的语言，所以编程也就和文字的重要性差不多。

三千多年前能识字写字的人，可能是写下《汉谟拉比法典》中的一员。两千年前会识字写字的人，可能有机会坐在孔子面前听课。300年前会识字写字的，在周围人中多少算个人物。

现在会识字写字的人，什么都不是。但谁要是不能识字写字，反倒成了少数人了，很多基本的事都干不了，就连智能手机玩起来都困难。

编程的历史很短暂，比文字的历史短暂多了。

五六十年前懂编程，运气好点可能就是比尔盖茨之类的大富豪，运气差点也可能是微软等大公司员工编号前1000的员工。现在懂编程，运气好点，可以去大厂当个程序员，运气差点也可以是自己计算机的高级用户。今后孩子们长大了懂编程，什么都不是。但是，到了那个年代，不懂编程的人绝对是文盲，很多很多最基本的事情都做不了。

除了这些大趋势，编程对孩子短期的好处很多。

短期的直接好处​

高考要考

浙江省一直是中国高考的排头兵，走在教育改革的最前沿。以我熟悉的英语为例，2017年浙江高考开始增加了读后续写这个题型，目前全国已经有一多半的省市高考英语要考了。2021年浙江高考又增加了一门科目叫做技术（包含通用技术和信息技术）。这门科目是除了语数外三门课以外，和物理化学生物历史地理政治处于同一高度的科目。

鉴于数字技术的重要性，以及浙江的先行试点，相信过不了十年，到不了土豆参加高考的时候，重庆也会和全国很多省市一样，高考中也会有技术这个科目。

如果孩子从现在就开始学习编程，肯定要比别人从高中才开始学习更有优势。（虽然现在小学也有一门科目叫做信息技术，但是那门科目在学校的重视程度是很低的）

小升初，初升高升学要用

又因为编程太重要了，懂编程的好处，要不了等到高考，就已经体现出来了。

在我们很多家长头痛的小升初，初升高择校时，就已经体现出来了。

在学生挑选好学校的同时，学校也在挑选好学生。每个学校的招生老师都想要招到更好的学生，除了要看学生的分数，还要看学生的后劲。

我们家长都相信自己家的孩子后劲十足，但学校招生老师却有他的看法。在同等分数，或者分数相差不大的情况下，很多招生老师都会优先选择有特长的学生。因为特长是用时间换来的，如果一个孩子在唱歌，跳舞，画画，足球，编程上有特长，招生老师有理由相信这个学生花在课本上的时间更少，却达到了和别人差不多的分数，那这个孩子就更可能是有后劲的学生。相比于在上一个阶段就已经“力竭”的学生，招生老师当然会优先录取有后劲的学生。

因为编程直接就用到了很多数学，英语的内容。同样是有特长，招生老师就会更加倾向于有编程特长的学生，而不是有唱歌，画画特长的学生。

搞不好，你让孩子学习编程， 在择校方面的优势还大于买好几百万的学区房。

甚至，学习编程的好处也不用等到孩子升学的时候，在小学就已经有充分的体现了。

以土豆所在的学校为例，除了学校要开展信息技术这门科目，学校更重要的方式是开设信息技术的各种社团。在社团中孩子们学习编程等技术，并运用到具体的项目中，包括机器人控制，无人机控制等等项目，让孩子们在实践中学习编程。学校还要组织孩子参加各种信息技术的比赛，他们学校也有获得过全国性的冠军。

这样的社团肯定是有名额限制的，参加比赛，更是难得的机会。最终这些机会都会偏向于有准备的孩子。

编程还给了孩子以下多种的好处：​

  1. 提升逻辑思维能力：编程需要孩子们按照一定的逻辑和顺序来组织代码，这有助于培养他们的逻辑思维能力。
  2. 增强解决问题的能力：编程往往涉及到解决各种问题，例如修复错误、优化代码等。这些都能锻炼孩子的问题解决能力。
  3. 培养实践能力：编程是一项实践性很强的技能，通过编程，孩子们可以将理论知识转化为实际的产品或项目。
  4. 提高创新意识和创造力：编程可以让孩子们创造出自己的游戏、动画、网站等，这有助于提高他们的创新意识和创造力。
  5. 学习现代科技知识：在这个信息化的时代，掌握编程知识可以让孩子更好地理解和应用现代科技。

2.土豆的编程学习过程​

因为认识到了编程的重要性，我就给孩子报了少儿编程的课。​

选择的也是大机构，上市公司的编程课。然而，上市大公司的编程课，却并没有宣传起来的那么好。

一周一次课，一次课一个半小时。然而，这样的编程课，不仅仅是频率太低，学的东西还没有忘的多，更重要的问题是上课几乎只是老师带着孩子在玩积木，和编程扯得上关系的无非是刷一下早已制作好的成品卡片。

按照他们的课程规划，一直要学4-5年的实物编程和图形化编程，要到10岁以上才开始纯文本的编程。虽然机构宣传这样更科学，但我多少怀疑这是没必要的拉长时间。

又加上接送孩子去上课太麻烦，我在边上等的实在太无聊，我放弃了……

最终我在机缘巧合之下，今年给孩子找到了Swift Playground 这个神器。​

Swift Playground 是 Apple 开发的一种编程学习环境，它允许学习者使用 Swift 这款苹果IOS系统的编程语言进行学习和实践。Swift 本身就是一种现代、强类型的编程语言，由 Apple 开发，专为 iOS、macOS、watchOS 和 tvOS 设计。

苹果的开发实力，也确实不是什么少儿编程培训机构所能比拟的。Swift Playground 提供了一个交互式的环境，学习者可以在其中编写 Swift 代码并即时查看结果。虽然它是基于文本的编程学习环境，但它采用了可视化的方式呈现代码、输出和结果，使得编程学习更直观、有趣和容易理解，特别适合初学者。

在 Swift Playground 中，学习者可以通过编写文本代码来创建程序、解决问题，以及学习编程的基本概念，而且这些代码是以文本形式呈现的，而不是以图形块的形式。

土豆学了Swift Playground一段时间后，我惊讶地发现，这样的内容，小孩子原本就是看得懂，学得会的。​

根本不需要用线下课那缓慢的进度来开展。相比之下，土豆之前去上的线下编程课，就真的弱爆了，完全是在浪费时间。

明明孩子可以用一年的时间就能学会的内容，在线下5年都可能学不会。

现在土豆已经学习了2个月左右，已经通关了《编程入门》和《学习编程1》这两个项目，正在学习《编程学习2》这个项目，已经掌握了一些编程的基础知识。

而学习Swift Playground也非常简单，只要一个iPad，直接下载这个软件，就可以了。

虽然土豆现在才开始学习编程，也并不清楚后面会怎样。但我知道，这一定比送孩子去线下的培训班要好。即使孩子遇到的问题我和他不能解决，也有很多高手可以帮助我解决。我对这条路，充满了信心。

3.抱团学习编程​

社群的重要性​

我自己在学习编程的社社群里，我知道社群的重要性。我也体验过一个人可以走得更快，一群人才能走得更远。现在如果我只教土豆一个人的话，确实土豆可以学得快一点，但是我相信他的编程注定走不了多远的。最终我们会遇到一些问题，然后就卡住了，然后就停了下来。如果我们是一群孩子在学习编程的话，他会看到其他孩子遇到的问题是怎么通过持续的努力而解决，进而继续开展下去的。这样的经历，对他自己也会有更大的帮助。土豆才能在他遇到问题后，还能继续走下去。

身边很多的朋友看到土豆在学Swift Playground后，也希望自己的孩子来学习，但是由于自己的工作时间不灵活，以及自己目前还没有多少编程知识，就很难开展下去。

其中一些朋友就希望我能帮忙​

鉴于土豆的经历，以及我们培训英语的成功经验，我对于教小孩学习编程有信心，可以来尝试一下。

因此，我决定自己组建一个学习社群，教和土豆差不多大的孩子学习编程。我怎么教土豆，就怎么教你的孩子。不会像外面的少儿编程课程，非要把一年的内容拖到五年。

有需要了解如何帮助孩子学习编程的朋友，可以联系我。
