奖励刺激，打骂处罚都管不久，可以这样提高孩子学习兴趣，信心和执行力

我和土豆妈妈都认为语文和数学是非常重要的，但是学校的课堂是承载不了如此重要的内容，学校考试的分数，更无法来衡量语文和数学这样重要的能力的。

因此我们不看重学校的分数，也不想让孩子在学校的内容上花太多的时间，所以我们一直没有让孩子上过学校的延时课，也没有给孩子额外布置练习题，更没有给孩子补过相关内容。考试分数吧，不拉班级后腿就行了，别给老师添麻烦。我们也有和老师沟通，老师也认可我们的做法。

虽然土豆没有上延时课，没有去补课，但是土豆的学习内容并不少。

每天土豆放学后，除了做学校的作业，要运动30分钟到一个小时，要阅读课外非虚构类书籍半小时左右，要自学英语半小时左右，自学编程半小时左右，还要复习之前学过的内容。

无论是学校的内容，还是土豆自学的内容，他有较强的学习意愿，不需要我们家长费力地去推着他走。

家长都希望自己孩子多学习，毕竟在青少年时期尽可能地多学习一切可以学习的内容，是孩子的本分。

为了孩子多学习，有的家长奖励刺激孩子，也有的家长打骂处罚孩子，这些短时间内可能有用，但是时间一长，效果就不大好。

如何让孩子自己想学习，爱学习呢？如何提高孩子学习的执行力，自信心呢？如果孩子对学校的内容都已经不胜厌烦了，如何让孩子还有心思去学习家长认为该学的内容呢？

这些确实是摆在我们家长面前的一个难题。

我介绍一下土豆学编程的过程，或许对大家有用。

土豆学编程的四个阶段：​

阶段一：父母要我学，我就学​

因为我认为编程很重要，就在土豆上幼儿园的时候给土豆报了线下的编程课。

上课的形式就是老师带领孩子们玩积木，把积木组装起来，最后刷一下带有特定程序的卡片，组装的积木就进行相应的运动。有老师带着玩，土豆当然也不抗拒。但后来我告诉土豆咱们不学这个了，土豆也没有反对，反正每天好玩的这么多……

在这个过程里，是家人让他学，他就学，是完全被动的学习。

阶段二：讲道理，反复讲道理，变着花样反复讲道理，知道了该学。​

后来因为我对编程的知识增加了，我就能给孩子讲清楚编程为什么重要了。

我就反复地给土豆讲编程为什么重要。

讲编程对我们长期有什么帮助，对我们今后工作有什么帮助，对我们思维有什么帮助。

也讲编程对我们短期有什么帮助，对在学校学习有什么帮助，对后面升学有什么帮助。

（具体内容见这篇文章）

不仅讲了学好了有什么好处，还主要讲没学会有什么后果……

还给土豆讲，学编程为什么不能指望学校，既不能指望中小学，也不能指望大学。编程在中小学不够重视，到了大学对应的专业也教不了。因为计算机科学进步得太快了，80%的计算机科学的论文都不是大学和科研机构写出来的，大学里的教授甚至都看不懂。

反正只有一条路，自己自学，早晚都得靠自学。

这个过程不是一次，而是很多次。也不是一两天，而是很多天。讲的内容不相同，角度也不相同，反复换着花样讲。

土豆被灌输的时间长了，听的角度多了，就自然认同了该学习编程，有了想学编程的想法。

在这期间，一向节约的土豆，甚至拿出了50元巨款的零花钱，向我交了学费，要我教他编程。

这时初步目标已经达成，土豆已经自己想要学习编程了。

但我认为时候还没到，于是我告诉土豆最近我很忙，过两年再教，土豆也还能接受。

阶段三：看着爸爸用编程做的游戏，很想要学​

这两年我在学习编程，无论是看书的过程，还是用编程做东西的过程，我都会展示给土豆。无论是我遇到了困难，还是取得了进展，整个过程，土豆都清楚。前后学习了用GPT做一个中英文同声传译对话的小程序，学习做了一个帮助我实现在线翻译英语书籍的网址，做了一个调色板的APP，在这期间，土豆都是我做出内容的首席体验官。

直到土豆看到我敲出好几个游戏，土豆对编程的兴趣到达了顶点。

在今年的暑假，我照着一本编程的教材，敲出了一个《抵抗外星人》的游戏，从最开始只有一架飞机，到增加一个外星人，再到整个外星人编队，再到整个游戏能运行还能记分，土豆一直看在眼里。哪个孩子不想玩自己做出来的游戏呢，再说了，爸爸不就是在这里一直敲键盘吗，还是照着书上来敲的键盘，这有什么难的。

在这段时间内，土豆天天找我教他编程。

以前土豆是知道该学习编程的，也是想要学编程的。但是知道该学，和看到了我做到了学编程的用处，对他的吸引是完全不可同日而语的。在看到我用编程敲出游戏后，土豆对学习编程已经变成无法等待的急切渴望了。

这一次，我再也无法推脱了。

我告诉土豆，这个事情不是学几次，也不是学几天，就可以了。而是需要学习好多年，甚至一直学下去，也一直受益的事情。在这种情况下，土豆还是非常愿意来学习。于是土豆的编程学习，就正式开始了。

从今年暑假的8月份到现在，土豆已经学习了两个多月的编程了。

阶段四：做到在Playgrounds中闯关后，停不下来​

现在土豆每天用半个小时左右学习编程，这个过程不需要我催促，他的积极性很高。无论是周中还是周末，都想要学习。由于Swift Playgrounds是面向初学者的，以游戏化的内容来开展教学，在玩游戏的过程中学习编程。这一段时间以来，土豆通了很多关，从《编程入门》到《学习编程1》，再到现在的《学习编程2》。在这个学习期间，也遇到了很多的困难，每一次被难住，都没有想过放弃，而是自己反复尝试。实在自己做不出来，再来找我帮忙。无论是他独立通关，还是在我的帮助下通关，他都开心得不得了。

相比于上一个阶段看到别人做到，这个阶段里，土豆自己做到了一部分内容，对他的激励更大。

现在在他的心理，他不会认为编程是困难的，而认为是简单的。再说了，我们也不指望孩子短时间就达到一个什么样的水平，这样重要的技能，这样具有繁衍性的知识，持续学习下去就好了，学得越久越好。

现在对于学编程，已经成了谁不让他学习，他就会跟谁着急……

在这情况下，根本不需要家长去奖励他，处罚他，根本就不需要家长的刺激，他自己就会去学习……

提高学习兴趣，以下三点可以起到作用：​

1.反复变着花样地讲道理，灌输学这个内容的好处，和没学会的坏处。​

所有的知识技能都有用，但是有的知识技能的重要性更高。我们让孩子运动，阅读，学英语，学编程，而不是学习美术，音乐，书法，跆拳道……，是因为这几样相对更重要。

因为这些内容非常有用，我们就可以反复变着花样的灌输学这个内容的好处。

因为不需要和别人比，我们就可以慢慢学，可以用很长的时间来学，孩子的学习就没有压力。

学编程有这么多学编程的好处，学英语也同样有很多很多的好处，学其他什么内容，也都反复变着花样地给孩子灌输学习的好处，也给孩子讲出没做到有什么后果，给出孩子无法拒绝的理由。

2.让孩子亲眼看到别人做到，并和做到的这些人处在一个社群。​

想要孩子做到，就要让孩子先接近达到了这个目标的人群。在这样近距离的观察下，孩子会看到的具体的过程，而不是最终包装好的产品结果。孩子会看到，这些人到底是怎么做的，他们遇到了什么困难，遇到了困难时是怎么应对的……

孩子会明白，那些做到了的人，也和我一样，没有什么特别之处。只是因为他们过往做到了相应的积累，现在他们就能做到。甚至这些做到了的人，在某些方面，还比不上我呢。

孩子就会相信，我只要做到了正确的积累，我也能做到的。

这里有一个注意事项：

有一些培训机构，为了宣传他们的课程，会包装他们的老师，说他们的老师是多么厉害，是毕业于什么学校，是来自什么大厂的程序员。这样对于他们的销售有帮助，但对孩子学习编程并不好。首先，在当今这个时代下，大厂优秀的程序员是不可能放着高薪收入不要，来为培训机构卖命教小孩子……

更重要的是，这样不真诚的宣传下，会让孩子误以为他是专业的，他很厉害，他才能做到。而我是平凡普通的，所以我做不到……

3.设计一条学习难度平缓的曲线，让孩子逐步做到。​

孩子抱着一腔热情来开始学习了，不能一下子就把孩子难倒，否则就是一桶冷水浇在孩子头上，热情立马就熄灭了。让孩子在学习过程中，应该帮助孩子及时获得正向反馈。遇到的难点卡点也能及时被拔出掉，不能因为一个知识点过不了，就一直被拦在这里。

假以时日，孩子通过学习成长了，孩子的积累多了，即使遇到某些困难，即使在这里走不通，也能从其他地方绕过去，也就能继续学习下去。

后续做点什么：​

为了让土豆的编程能走得更远，我需要让他处于社交学习中，因此需要帮土豆找到同龄的孩子一起学习编程，这样土豆的编程之路才可以走得足够久，才能走得足够远。另外身边一些不懂编程的家长朋友，也希望我能帮到他们的孩子。因此，我有个想法是大家一起抱团取暖，大家一起来学习编程。

如果家长不清楚编程对孩子有什么好处，不能帮孩子看到编程的作用，不清楚如何帮助孩子学习编程，我也可以尽到微薄之力。

与外面机构最大的不同是，我会很真诚，我不会在孩子面前装得很厉害。我会给孩子们展示，他身边的一个普通人，像土豆爸爸这样的一个普通人，在三十多岁了，知道了编程很有用以后，从原来完全不会，到现在编程变得对他很有用了。孩子们会相信，这个真的不难的，而且这个真的应该学。他们现在就开始学，比起土豆爸爸等到几十岁才学习，优势更多也就更有信心，时间更多也就更从容……
