        • 发现一个问题，自己总是不能专注。不能做到要事第一。现在想知道，保持好奇与保持专注的边界在哪里？现在想到的一个想法是，有新的想法时，先不忙行动，先写下来。再一个，可以判断这个内容能不能做得久？值不值得做得久，做得久投入的成本我能否承受[[好奇]][[长期]]
        • 后面关注笑来的课程：专注的真相，看看能不能从这个课程中找到答案。
        • 关于信任，信用，说话算话，不要轻诺寡信。土豆最近答应了出去玩半个小时，但是却是3个小时以后才回来。[[孩子]] [[故事]][[写作]]
          • 土豆出去玩，总是不按时回来，给他讲讲信用，在长打交道的人面前失去信用的后果，维持信用的成本，不要轻诺寡信，不要轻易承诺和许诺，许诺之前先想一想自己能不能做到，在哪些情况下会很困难做到。答应了以后，就要尽量做到，千方百计做到。实在没做到，要给别人解释清楚。这个对其他孩子也有用。可以用将故事的形式讲出来[[故事]]
          • 失信于人的后果，失信于家人的后果：
            • 小叮当，原本说好的只玩半个小时，到了时间不会去，妈妈下一次可能就不让她出来玩了。妈妈时间充足的时候才会让她出来。原本说话算数的话，只有半个小时及以上的情况下，妈妈都会让她出来玩。现在说话不算话，只有少数有一个小时以上的情况下，妈妈才会让她出来玩。小叮当虽然每次和妈妈争着多玩一会儿，却错过了很多出来玩的机会。
            • 说好不玩手机游戏的，结果要玩，还偷偷玩，本来要用ipad学编程的，妈妈不让她学了，妈妈害怕她控制不住自己。小叮当赢得了偷偷玩手机的时间，却错过了用IPad学编程的机会。
            • 说好要认真做作业的，结果每次磨洋工。妈妈没有足够的精力来照顾小叮当的学习了，妈妈只能送她去很远的地方上学，不能在家附近的学校上学。小叮当赢得了磨洋工，却错过了在盛德沙小上学的机会。
            • 你看，失信于人的后果很严重吧，虽然这些后果有时候不是直接可见的，甚至因为有滞后性，看不清楚是因为失信与人导致的。尤其是失信于长期和你反复打交道的人。当然，维护自己的信用也是需要成本的。
              • 成本一：不能轻易承诺，随便许诺。轻易许诺能在当时讨好别人。但事后会被别人看穿，别人也会采取相应的对策。
              • 成本二：承诺后的兑现。到了兑现承诺时，有时候是做不到，有时候是不愿意。
          • 对策：
            • 不要轻易承诺，做不到的就不要答应。承诺之前，先想想有哪些情况会是我执行的时候很困难，到底能不能做到。不要只图当时承诺时的好处，就轻易承诺。更不要轻易去讨好别人。
            • 承诺以后，尽最大努力去做到。实在没做到，要道歉，要处罚自己，换取别人的信任。
          • 信用账户：
            • 人与人之间的信任，就像存钱一样，不断存进去，才能有强的信任。总是不守信，就是在透支自己的信用账户
            • 承诺即负债。承诺的当时，换来了当时别人的条件，换来了别人的好感，别人的欢喜。同时承诺，也欠下了别人东西，就像欠钱一样，欠了别人的债务。我们为什么会容易许诺：因为许诺可以换来别人的好感，别人的同意，在当时能帮我带来好处。比如家长问你能不能再半个小时内回来，你想要是说不能回来，家长可能不让你出去玩，所以你会说能。许诺在当时给我们带来了好处。
          • 交易成本的概念。
            • 没有信用，会导致交易成本上涨。交易往往能产生交易剩余即参与交易的双方都赚了，但是由于没有信任，就很难达成交易，交易达成量就下降了，就损失了那些原本可以因为交易而带来的收益。
          • 你提到了为什么不是家长给你打电话提醒你回来，这个就成了只有外部约束才履约，而我们应该自行主动履约
          • 好的我希望把这个事情给你讲清楚了，你有什么看法，下一次做到了或者没有做到，我们都可以交流。即使没有做到，也不要怕，我反复给你讲。
          • 我把道理反复给你讲，讲信用是道德的表现，不是期望你做一个道德高尚的人，而是讲清楚了后果，你就能知道讲信用，是对我们自己更有利的选择。

          • 直接执行部分：
            • 记录时用自己的话转述，复述，不能复制。——有一个地方，提到了复述
            • 记录想法，灵感，情绪，反直觉——情绪
            • 制作标签说明书，放在[[模板]]里面
            • 其中讲所有内容都带4个标签挺好的，其中一个是临时的，一个是兴趣，一个是精进，一个是掌握——这个还需要再看看
            • 临时：相当于一个缓存，自己来不及消化的，后续需要尽快处理的。处理后换成其他标签。
            • 兴趣：记录自己感兴趣的，对别人没有什么影响优先度不高
            • 精进：对自己很重要的，需要不断精进的区域，优先度最高
            • 近期：近期要做的事情
          • 这本书看完了，需要进行整理，能为我所用的部分
            • 记笔记，增援未来的自己，解决具体的问题。是为了未来的自己而写。
            • 预处理的方法：用自己的话复述转述
            • 笔记的运用：查看笔记能给自己新的想法，线索，依据
            • 必要难度理论，存入记忆越容易，提取越困难。所以，直接复制粘贴，或者语音转文字再粘贴，不是什么好方法。所以要用自己的话转述。这当中重要的是思考。用自己的话记笔记会倒逼思考。转述时感觉有难度，说明自己的理解可能还有问题。所以转述也就尤为必要。这也是笑来让大家听了课以后再讲给孩子听。
            • 用文字记录下来会倒逼自己思考。
            • 记录内容：
              • 记录想法：感动的内容，灵感，今后有用的信息
              • 记录有启发的内容：为什么感觉这个内容不错，将来自己能用在哪里，使用的边界在哪里，
              • 记录反直觉的内容：主动反过来想，做不到结果会是哪些原因，如果不做这个，会怎么样。转换视角。
              • 记录情绪波动的事情：记录情绪及情绪环境，好坏都要记录
              • 记录实战经验：为后续复盘做准备。比如我的销售
              • 反复整理，总结，归纳。要舍得删除后续没有用处的内容。这一点尤为必要。太多的内容会增加维护的成本，从而稀释好内容被复习的数量。
            • 笔记打标签，建立标签文件。
              • 确认是否有二级标签——没有
              • 制作标签说明书
            • 回顾笔记时，
              • 这条笔记和我最近遇到的事有什么关系
              • 引发了什么新的思考
            • 收集信息时：
              • 对自己没用的，不要做笔记，不要浪费太多时间在笔记上
              • 设置边界，不要什么都想去插一脚，专注于该精进的领域，自己待解决的问题
            • 以我为主，增援未来。持续不断
